# Naver Report Crawler 

**네이버 리포트 크롤러** 

## 🚀 서비스 소개
- 당일 네이버 증권에 올라온 개별 종목 증권사 리포트 크롤링 

## ✨ 주요 기능
- **개별 종목 리포트 크롤링**
  - 종목명, 증권사, 리포트URL
 
## 🛠 기술 스택
- Python

## ⚙️ 실행 방법
```bash

1. 저장소 클론 및 이동
git clone https://github.com/bomby06/naver-report-crawler.git

cd naver-report-crawler

2. 패키지 설치
pip install -r requirements.txt

3. 실행
python main.py
